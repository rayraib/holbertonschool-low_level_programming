# 0x09-argc_argv
### Synopsis
### *Holberton School * project to learn the following: What arguments are and how to use passed arguments to a program, how to use `__attribute__((unused))` or `(void)`
--------------------
### Tasks
| Tasks | Description |
| ----- | ---------- |
|**0-whatsmyname.c**| Prints program name, followed by a new line.|
|**1-args.c**| Prints the number of arguments passed into the program. |
|**2-args.c.**| Prints all arguments the program receives.|
|**3-mul.c**| Multiplies two numbers received as arguments from the command line.|
|**4-add.c**| Adds positive numbers received as arguments from the command line|
